# Minimalist Green Tea

## Descripción
Experience serene and fresh browsing in Google Chrome with Minimalist Green Tea theme.

## Autor
Miguel Euraque

## Instrucciones de instalación
1. Descarga el archivo \`.zip\` del tema.
2. Ve a \`chrome://extensions/\` en tu navegador Chrome.
3. Activa el "Modo de desarrollador".
4. Haz clic en "Cargar descomprimida" y selecciona la carpeta del tema.
